This repository documents my hands-on AWS Cloud learning journey.

I am completing beginner-friendly AWS labs to practice cloud fundamentals, AWS services, security basics, troubleshooting, and CloudOps skills.

## Goals

- Build practical AWS experience
- Document each lab clearly
- Practice AWS CLI and cloud administration
- Learn cloud security best practices
- Build a GitHub portfolio for cloud and cybersecurity roles
